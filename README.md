# bnf.net

A .NET library for working with BNF (Backus-Naur Form) grammars.

## Projects

- **bnf.net** - Core class library
- **bnf.net.Demo** - Sample console application demonstrating library usage
- **bnf.net.Tests** - Unit tests

## Building

```bash
dotnet build
```

## Testing

```bash
dotnet test
```
